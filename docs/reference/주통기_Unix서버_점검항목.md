| 항목코드 | 항목명 | 중요도 | 점검 내용 |
|---------|--------|--------|----------|

| U-04 | 패스워드 파일 보호 | 상 | /etc/passwd, /etc/shadow 권한 설정 |
| U-44 | root 이외의 UID가 0인 계정 금지 | 상 | UID 0인 계정이 root만 존재하는지 확인 |
| U-45 | root 계정 su 제한 | 상 | su 명령어 사용 가능 그룹 제한 (wheel 그룹) |

| U-48 | 패스워드 최소 사용 기간 설정 | 상 | 1일 이상 설정 여부 |
| U-49 | 불필요한 계정 제거 | 중 | 시스템 미사용 계정 삭제 여부 |
| U-50 | 관리자 그룹에 최소한의 계정 포함 | 중 | 관리자 그룹(root, wheel) 최소 인원 |
| U-51 | 계정이 존재하지 않는 GID 금지 | 중 | orphaned group ID 제거 |
| U-52 | 동일한 UID 금지 | 중 | 중복 UID 존재 여부 |
| U-53 | 사용자 shell 점검 | 중 | 시스템 계정이 /sbin/nologin 사용 확인 |
| U-54 | Session Timeout 설정 | 중 | 유휴 세션 자동 종료 시간 설정 (300초 이하) |
